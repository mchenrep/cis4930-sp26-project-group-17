## Github Basketball Related Repositories 
This project collects important data about basketball related repositories on Github. These include popularity (amount of stars), language, and any contributers/creators of repositories. In a real world context, these would be good metrics to track for insight into basketball related projects or collaboration with a user/repository.

## API Documentation
[Github API Documentation](https://docs.github.com/en/rest)

## Data Pipeline Goals
1. Fetch 10 pages of repositories per run
2. Store results in an CSV File or SQLite database
3. Handle failures and log errors
